# 📄 MergeMate

A modern, fast and privacy-focused PDF Toolkit built with JavaScript.

> 🔒 No Upload • No Watermark • Free Forever

---

## ✨ Features

- ✅ Merge PDF
- ✅ Split PDF
- ✅ Delete Multiple Pages
- 🚧 Extract Pages
- 🚧 Rotate PDF
- 🚧 Images to PDF

---

## 🛠 Built With

- JavaScript (ES6)
- Vite
- pdf-lib
- HTML5
- CSS3

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/vishalxcodes/MergeMate.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
